# Timetable Project

**Timetable Project** is a Python desktop application built with **Tkinter** that helps students manage their course schedules and save selected courses to a CSV file.

- Prevents schedule conflicts and warns users if conflicts exist.  
- Saves the selected timetable to a CSV file.  
- User-friendly GUI designed with **Tkinter**.
